# Beyond Good & Evil Akuda Bar Locker Code Decoder

A TypeScript conversion of [JDarkroom](https://github.com/TheEntropyShard/JDarkroom) by [TheEntropyShard](https://github.com/TheEntropyShard),
with some small optimizations, designed to do the processing locally, in the user's browser.
Made because I dislike having to install the JRE and I didn't find https://darkroom.bgemyth.net/ on my first search.

### Usage

Use it online at https://fragilematter.github.io/bge_akuda_ts/
All processing happens in your browser, there are no external dependencies, no information is sent to other servers.

Locally, make sure you have npm installed, then run
```console
    npm run build
    npm run serve
```
